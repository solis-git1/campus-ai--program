# 校园助手小程序 - 用户端接口文档

## 📋 文档概述

本文档详细说明校园助手小程序用户端的所有 API 接口，包含用户管理、课程管理、活动管理、教室管理、待办任务、AI推荐、系统通知、AI助手等核心功能模块。

## 📊 接口统计

- **用户管理模块**: 4 个接口
- **课程/课表模块**: 5 个接口
- **校园活动模块**: 5 个接口
- **教室管理模块**: 5 个接口
- **待办任务模块**: 4 个接口
- **AI智能推荐模块**: 2 个接口
- **系统通知模块**: 2 个接口
- **AI助手模块**: 2 个接口
- **总计**: 29 个接口

## 🔧 技术规范

### 基础信息
- **基础路径**: `/user`
- **认证方式**: JWT Token
- **请求头**: `Authorization: Bearer {token}`
- **数据格式**: JSON

### 通用响应格式
```json
{
    "code": 200,
    "message": "success",
    "data": {}
}
```

## 1. 用户管理模块 (UserController)
**基础路径：** `/user/user`

### 1.1 微信小程序登录接口
- **请求方式**: POST
- **接口路径**: `/user/user/login`
- **功能描述**: 微信小程序登录接口，支持新用户自动注册
- **请求参数**:
  ```json
  {
    "code": "微信授权code"
  }
  ```
- **响应数据**:
  ```json
  {
    "code": 200,
    "message": "success",
    "data": {
      "userId": 123,
      "token": "jwt_token",
      "nickname": "用户昵称",
      "avatar": "头像URL"
    }
  }
  ```

### 1.2 用户注册接口
- **请求方式**: POST
- **接口路径**: `/user/user/register`
- **功能描述**: 用户注册接口（账号密码注册）
- **请求参数**:
  ```json
  {
    "username": "用户名（3-20位字母数字下划线）",
    "password": "密码（最少6位）",
    "nickname": "昵称",
    "phone": "手机号（可选）"
  }
  ```
- **响应数据**:
  ```json
  {
    "code": 200,
    "message": "注册成功",
    "data": null
  }
  ```

### 1.3 获取用户信息
- **请求方式**: GET
- **接口路径**: `/user/user/info`
- **功能描述**: 获取当前登录用户的信息
- **请求头**: `Authorization: Bearer {token}`
- **响应数据**:
  ```json
  {
    "code": 200,
    "message": "success",
    "data": {
      "userId": 123,
      "username": "用户名",
      "nickname": "昵称",
      "avatar": "头像URL",
      "phone": "手机号",
      "createTime": "2024-01-01 00:00:00"
    }
  }
  ```

### 1.4 更新用户信息
- **请求方式**: PUT
- **接口路径**: `/user/user`
- **功能描述**: 更新用户信息
- **请求头**: `Authorization: Bearer {token}`
- **请求参数**:
  ```json
  {
    "nickname": "新昵称",
    "avatar": "新头像URL",
    "phone": "新手机号"
  }
  ```

## 2. 课程/课表模块 (CourseController)
**基础路径：** `/user/course`

- **请求方式**: GET
- **接口路径**: `/user/course/my`

  ```json
        "endTime": "09:40:00",
        "type": "必修",
        "note": "重要课程",
        "status": "current"

    ]
  }
  ```

### 2.2 获取课程详情
- **请求方式**: GET
- **接口路径**: `/user/course/{courseId}`
- **功能描述**: 获取课程详情
- **请求头**: `Authorization: Bearer {token}`
- **路径参数**: `courseId` - 课程ID

### 2.3 添加课程到我的课表
- **请求方式**: POST
- **接口路径**: `/user/course/add/{courseId}`
- **功能描述**: 添加课程到我的课表
- **请求头**: `Authorization: Bearer {token}`
- **路径参数**: `courseId` - 课程ID
- **查询参数**: `note` - 备注信息（可选）

### 2.4 从我的课表删除课程
- **请求方式**: DELETE
- **接口路径**: `/user/course/remove/{courseId}`
- **功能描述**: 从我的课表删除课程
- **请求头**: `Authorization: Bearer {token}`
- **路径参数**: `courseId` - 课程ID

### 2.5 设置课程提醒
- **请求方式**: POST
- **接口路径**: `/user/course/reminder/{courseId}`
- **功能描述**: 设置课程提醒
- **请求头**: `Authorization: Bearer {token}`
- **路径参数**: `courseId` - 课程ID
- **查询参数**:
  - `remindTime` - 提醒时间（格式：yyyy-MM-dd HH:mm:ss）
  - `repeatType` - 重复类型（默认：week）

## 📝 使用说明

### 认证流程
1. 调用 `/user/user/login` 接口获取 token
2. 将 token 添加到后续请求的 Authorization 头中
3. token 过期后需要重新登录

### 错误处理
- **401**: token 过期或无效，需要重新登录
- **403**: 权限不足，检查用户角色
- **404**: 资源不存在，检查参数是否正确
- **500**: 服务器错误，联系技术支持

### 分页说明
- 所有分页接口都支持 `page` 和 `pageSize` 参数
- 默认 page=1, pageSize=10
- 返回数据包含分页信息
