## 📋 文档概述

本文档详细说明校园助手小程序管理员端的所有 API 接口，包含管理员管理、用户管理、活动管理、课程管理、教室管理、数据统计、推荐管理、AI会话管理、知识库管理、通知管理、日志管理、系统管理等后台管理功能模块。

## 📝 使用说明

### 认证流程
1. 调用 `/admin/admin/login` 接口进行登录
2. 登录成功后，服务器会设置 Session
3. 后续请求会自动携带 Session 信息
4. 长时间未操作需要重新登录

### 权限说明
- **超级管理员**: 拥有所有权限
- **普通管理员**: 拥有部分管理权限
- **用户管理员**: 仅能管理用户相关功能

### 分页说明
- 所有分页接口都支持 `page` 和 `pageSize` 参数
- 默认 page=1, pageSize=10
- 返回数据包含分页信息

### 数据操作规范
- **创建操作**: 使用 POST 请求
- **查询操作**: 使用 GET 请求
- **更新操作**: 使用 PUT 请求
- **删除操作**: 使用 DELETE 请求
