## 🔧 技术规范

### 基础信息
- **项目名称**: 校园助手小程序
- **技术栈**: Spring Boot + MyBatis + MySQL
- **认证方式**: JWT Token（用户端）/ Session（管理端）
- **数据格式**: JSON

### 环境信息
- **开发环境**: `http://localhost:8080`
- **生产环境**: `https://api.campus.example.com`

### 通用响应格式
```json
{
    "code": 200,
    "message": "success",
    "data": {}
}
```

### 错误码说明
- **200**: 成功
- **400**: 请求参数错误
- **401**: 未授权/登录过期
- **403**: 权限不足
- **404**: 资源不存在
- **500**: 服务器内部错误

## 📖 使用指南

### 快速开始
1. **查看接口概览** → [API接口文档.md](API接口文档.md)
2. **查看详细说明** → [详细接口文档.md](详细接口文档.md)
3. **按模块查看** → 查看对应模块的接口说明

### 开发建议
1. **用户端开发**: 重点关注用户管理、课程、活动、教室等核心模块
2. **管理员端开发**: 重点关注数据统计、用户管理、系统管理等功能
3. **测试验证**: 使用提供的接口示例进行测试
